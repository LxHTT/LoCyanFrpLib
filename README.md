## LoCyanFrpLib
<p align="right">presented by LxHTT</p>

这是一个封装了LoCyanFrp API的Python库。

### 安装
___
```command
pip install LCFLib
```

### 开始使用  
___
#### 导入  
```python
import LCFLib
```
#### 设置是否绕过系统代理  
```python
from LCFLib import BYPASS_SYSTEM_PROXY

BYPASS_SYSTEM_PROXY(True) # True为开启绕过, False不绕过
```
...

### 提示
API分v1（未完工）与v2，如`LCFLib.v2`
